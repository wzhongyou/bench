# Contributing

## Getting Started

1. Fork and clone the repo
2. Open `Bench.xcodeproj` in Xcode
3. Make your changes on a feature branch

## Conventions

- All code, comments, docs, and commit messages in English
- Follow Swift API Design Guidelines
- Use SwiftUI for UI, AVFoundation for media processing
- Keep v1 scope focused — Mac only, local only, AI-assisted editing only

## Before Submitting

- Ensure the project builds without errors
- Run the test suite: `xcodebuild -project Bench.xcodeproj -scheme Bench test`
- Keep changes minimal and focused
