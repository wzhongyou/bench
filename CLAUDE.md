# Bench

macOS video editor with deep AI integration.

## Language

- All code, comments, docs, and commit messages in English
- README_zh.md is the only exception (Chinese mirror for users)

## Tech

- **Language**: Swift
- **UI**: SwiftUI
- **Video**: AVFoundation / VideoToolbox
- **Storage**: SQLite (GRDB)
- **Build**: Xcode + SPM
- **Target**: macOS 14+
